# My Crêpe — Argenteuil 🥞

Site vitrine de la crêperie **My Crêpe**, 136 rue Paul Vaillant Couturier, 95100 Argenteuil.

Crêpes salées & sucrées à composer, paninis, croque-monsieur, smoothies & milkshakes. Halal · Ouvert jusqu'à 04h · Livraison.

## Structure

| Fichier | Rôle |
|---|---|
| `index.html` | Page d'accueil |
| `menu.html` | La carte complète avec les prix |
| `css/style.css` | Design (couleurs et polices modifiables en haut du fichier) |
| `assets/` | Photos du restaurant |
| `LISEZ-MOI.txt` | Guide d'édition pas-à-pas |

## Voir le site

Ouvrir `index.html` dans un navigateur. Site statique, responsive (mobile + ordinateur).

## À confirmer avec le client

- Prix du menu (estimations à valider)
- Téléphone exact, horaires
- Comptes Instagram / Snapchat / TikTok
- Logo officiel et vraies photos
